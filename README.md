# Daily Expense Tracker System

## Overview

Daily Expense Tracker System is a full-stack web application developed using Spring Boot and MySQL that helps users manage their personal finances efficiently.

The application allows users to:

* Track daily expenses
* Manage monthly budgets
* View expense analytics
* Upload profile pictures
* Monitor expenses using charts
* Access admin dashboard features

---

## Features

### User Features

* User Registration & Login
* Session Management
* Add/Edit/Delete Expenses
* Search & Filter Expenses
* Monthly Budget Management
* Dashboard Analytics
* Expense Category Pie Chart
* Profile Image Upload

### Admin Features

* Admin Login
* View All Users
* Activate/Deactivate Users
* Total User Analytics
* System Expense Monitoring

## Setup Instructions

1. Clone repository
2. Open project in IntelliJ IDEA
3. Configure MySQL database
4. Update application.properties
5. Run Spring Boot application
6. Open:
   http://localhost:8080
